# Conduit knowledge

A Conduit component for knowledge functionality

## Installation

```bash
# Via Conduit component system
conduit components install knowledge

# Via Composer (if published)
composer require jordanpartridge/conduit-knowledge
```

## Commands

- `conduit init`
- `conduit status`
- `conduit configure`
- `conduit list`

## Usage

```bash
# Example usage
conduit init --help
```

## Development

```bash
# Install dependencies
composer install

# Run tests
./vendor/bin/pest

# Code formatting
./vendor/bin/pint

# Static analysis
./vendor/bin/phpstan analyze
```
